# chinese-china-journal

独立 HTML 页面归档仓库。

## 简介

本仓库用于收集、整理并发布多个独立的 HTML 页面。这些页面内容涉及多种主题，均以静态 HTML 文件的形式存放，便于直接浏览与存档。

仓库不针对任何特定域名或网站，仅作为页面归档与展示的通用空间。

## 目录说明

```
├── pages/          # 存放所有独立 HTML 页面
│   ├── example.html
│   └── ...
├── assets/         # 页面引用的公共资源（CSS、图片等）
├── index.html      # 仓库首页，可展示页面列表
└── README.md       # 本文件
```

- `pages/`：所有归档页面的主目录，每个文件为一个独立 HTML。
- `assets/`：如果页面需要共用样式或图片，可放置于此。
- `index.html`：可选，用于生成页面索引或导航。

## 页面归档说明

- 每个 HTML 页面均为自包含的静态文件，原则上不依赖外部资源（除本仓库内的公共资源外）。
- 页面内容可能涉及多种话题，不代表任何特定立场或主张。
- 归档仅作保存与展示用途，不保证内容的时效性、准确性或完整性。

## 维护说明

- 欢迎通过 Pull Request 提交新的独立 HTML 页面，或对现有页面进行修正。
- 提交前请确保页面为自包含的 HTML 文件，不包含恶意代码或非法内容。
- 仓库维护者保留审核、合并或拒绝提交的权利。
- 如有疑问或建议，请通过 Issues 进行讨论。
